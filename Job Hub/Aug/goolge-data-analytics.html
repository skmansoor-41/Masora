Strategic Cloud Engineer – Data Analytics

Company: Google

Location: Bengaluru or Hyderabad, India

Minimum Qualifications:

  • Bachelor's in CS, Engineering, Math, or related field
  • Experience with Python, Java, Scala, Spark, Hadoop
  • Hands-on with Google Cloud Platform
  • Data processing tools: BigQuery, CloudSQL, AlloyDB

Preferred Qualifications:

  • Big Data, Data Mining, ML experience
  • Experience with NoSQL, SparkML, TensorFlow
  • CI/CD & IaC tools (Terraform, Ansible, Jenkins)
  • Security concepts (HSM, encryption, key management)

About the Role:

As part of Google Cloud Consulting, you'll work on data migrations, analytics systems, and data governance with GCP tools. In this role, you’ll guide enterprises on cloud journeys, build scalable data pipelines, and conduct workshops to empower customers.

Responsibilities:

  • Translate requirements into scalable cloud architectures
  • Lead migrations to Google Cloud
  • Design and operationalize data systems
  • Implement data governance and quality controls
  • Work closely with customers, product, and engineering teams

Application: Visit Google Careers and search "Strategic Cloud Engineer - Data Analytics" or apply directly via your Google profile.
